# 铉枢 · 风格库

**XuanShu UI Gallery** — 6 种 Control Panel 主题风格，每种含独立 CSS + 可交互 Demo。

---

## 📖 风格总览

| 风格 | 预览 | CSS | 核心特征 |
|------|------|-----|----------|
| 🖌️ 水墨国风 | [Demo](https://tsingxuanhan.github.io/xuanshu-ui-gallery/styles/ink-wash/demo-inline.html) | 20.7KB | 宣纸纹理 · 朱砂点缀 · 墨晕背景 · 光扫动效 · 印章水印 |
| 🔮 玻璃拟态 | [Demo](https://tsingxuanhan.github.io/xuanshu-ui-gallery/styles/glassmorphism/demo-inline.html) | 19.8KB | backdrop-filter 毛玻璃 · 悬浮层次 · 渐变底色 |
| 🫧 新拟态 | [Demo](https://tsingxuanhan.github.io/xuanshu-ui-gallery/styles/neumorphism/demo-inline.html) | 21KB | 同色系双阴影 · 凹凸触感 · 内外交织 |
| 🟨 粗野主义 | [Demo](https://tsingxuanhan.github.io/xuanshu-ui-gallery/styles/neo-brutalism/demo-inline.html) | 16.9KB | 高饱和色块 · 硬阴影偏移 · 粗黑描边 |

## 🏗️ 仓库结构

```
xuanshu-ui-gallery/
├── index.html              ← 风格库总览页
├── README.md
├── original/               ← 原始 control-panel CSS 备份
└── styles/
    ├── ink-wash/           ← 水墨国风
    │   ├── style.css
    │   ├── demo.html       ← CSS 外联版
    │   └── demo-inline.html ← CSS 内联版（推荐预览）
    ├── glassmorphism/
    ├── neumorphism/
    ├── neo-brutalism/
    ├── minimal-swiss/
    └── cyberpunk/
```
