# 铉枢 · 风格库

> XuanShu UI Gallery — Control Panel Theme Collection

铉枢控制终端的 CSS 风格集合。每个风格独立目录，含完整 CSS + 可预览 Demo 页面。

## 风格列表

| 风格 | 目录 | 状态 | 特征 |
|------|------|------|------|
| 水墨国风 | `styles/ink-wash/` | ✅ 已完成 | 宣纸底·朱砂红·宋体·墨线 |
| 玻璃拟态 | `styles/glassmorphism/` | 🔜 待开发 | 毛玻璃·模糊·悬浮层次 |
| 新拟态 | `styles/neo-brutalism/` | 🔜 待开发 | 同色系浮雕·软阴影 |
| 粗野主义 | `styles/neo-brutalism/` | 🔜 待开发 | 高饱和·粗描边·零圆角 |
| 瑞士极简 | `styles/minimal-swiss/` | 🔜 待开发 | 极致留白·网格排版 |
| 赛博朋克 | `styles/cyberpunk/` | 🔜 待开发 | 霓虹·扫描线·Glitch |

## 使用方法

1. 本地打开 `index.html` 查看风格库总览
2. 点击任意风格卡片进入 Demo 预览
3. 每个 Demo 支持明暗主题切换

## 集成到控制终端

将目标风格的 `style.css` 复制到 `xuanshu-agents/control-panel/css/style.css` 即可替换。

## 原始版本

`original/` 目录保存了原始 Dark Glassmorphism 版本作为参考。
